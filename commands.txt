Optimized tool selection## Git Command List

- `git init` — create a new Git repository
- `git clone <repo>` — copy an existing repository
- `git status` — show working tree status
- `git add <path>` — stage changes for commit
- `git commit -m "message"` — record staged changes
- `git diff` — show unstaged changes
- `git diff --staged` — show staged changes
- `git log` — view commit history
- `git branch` — list branches
- `git branch <name>` — create a new branch
- `git checkout <branch>` — switch branches
- `git checkout -b <branch>` — create and switch branch
- `git merge <branch>` — merge another branch
- `git rebase <branch>` — reapply commits on top of another base
- `git remote -v` — list configured remotes
- `git remote add <name> <url>` — add a remote
- `git fetch` — download remote changes
- `git pull` — fetch and merge remote changes
- `git push` — upload commits to remote
- `git reset --hard <commit>` — reset branch to a commit
- `git reset HEAD <file>` — unstage a file
- `git rm <file>` — remove file from working tree and index
- `git stash` — save local changes temporarily
- `git stash pop` — reapply stashed changes
- `git tag <name>` — create a tag
- `git show <commit>` — show commit details
- `git blame <file>` — show who changed each line
- `git restore <file>` — discard working tree changes
- `git config --global user.name "Name"` — set global Git username
- `git config --global user.email "email@example.com"` — set global Git email
